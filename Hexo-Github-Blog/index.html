Hexo+Github = Blog

用工作空余时间终于搞出了基本功能，在这激动人心的时刻记录下过程，为第一篇博文

1

下载git和nodejs安装都是疯狂下一步

2

在本地电脑新建一个文件夹MyBlog作为博客后台（随便DEF盘），右键GitBashHere，第一次弄当然是先关联自己的github啦，没有GitHub的自己去申请一个

想了解更详细的git+GitHub操作和概念请戳这里英语好的戳这里
接下来添加SSH key

ssh -T git@github.com

npm install -g hexo //安装hexo
hexo -version //查看版本，有，说明安装成功啦
hexo init //初始化，这时作为博客后台的那个文件夹就有东西了
hexo g //生成
hexo s //启动本地服务，根据提示在浏览器输入localhost:4000，看到说明ok

Deployment

Docs: https://hexo.io/docs/deployment.html

deploy:
repository: 复制GitHub中clone or download的那个—>（git@github.com:<Github账号名称>/<Github账号名称>.github.io.git）
type: git
branch: master

npm install hexo-deployer-git –save //用以上传到GitHub
hexo d//上传到GitHub，过一会儿在浏览器输入:GitHub昵称.github.io看到的跟刚才一样

hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本
hexo clean # 清除缓存
hexo g -d # hexo generate和hexo deploy的组合生成和发布

git clone https://github.com/fan-lv/Fan.git themes/Fan
npm install hexo-renderer-jade –save //安装依赖啊
npm install hexo-renderer-stylus –save //安装依赖啊

hexo clean
hexo g -d
